# Aceita Fácil

Esta gem habilita as funcionalidades da API do serviço Aceita Fácil no Ruby on Rails.

## Installation

Adicione esta linha ao seu Gemfile:

    gem 'aceitafacil'

E execute:

    $ bundle

Ou instale assim:

    $ gem install aceitafacil
